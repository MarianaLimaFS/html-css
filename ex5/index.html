Usando links externos

Acesse meu repositório público no github

Usamos a tag <a> para envelopar palavras ou frases que você queira acoplar link. Utilizando os parametros "target _blank" e "rel external" permite que o link se abra em uma nova janela.

Usando links internos

Antes de criar um link interno, precisamos ter outras páginas, pois ele se refere a uma pagina propria. Para criar outra, só fazer um novo arquivo .html dentro da mesma pasta.
