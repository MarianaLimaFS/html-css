Parágrafos, emojis e imagens

Como funciona a criação de parágrafos e quebras de linhas:

É possível escrever um parágrafo de qualquer maneira desde que esteja dentro do par de tags <p> e </p>.

Para que haja uma quebra de linha de forma a criar outro parágrafo, é necessário inserir outro par de tags.

Adicionando emojis!

Procurar pelo site Emojipedia para ver o código do emoji desejado

Para adicionar o emoji é necessário digitar '&#x' antes do código do emoji e encerrar com ';'. 😊

Aproveitando esse html para iniciar o uso de imagens

Para inserir essa imagem, salvei na pasta, digitei a tag <img>, apertei a tecla 'ctrl' no teclado e a tecla 'espaço' e selecionei a opção da imagem que apareceu. Na tag, a sigla 'SCR' corresponde ao endereço da imagem e a sigla 'ALT' é onde posso dar uma legenda à foto.

Caso a imagem esteja numa pasta, é necessário indicar a pasta antes do endereço da imagem.

Para o carregamento de uma imagem de página externa, é preciso clicar com o botão direito na imagem desejada e copiar o endereço da imagem, depois, basta colar em 'scr'
